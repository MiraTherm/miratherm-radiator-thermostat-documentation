\section{Time plan}
\label{sec:Time plan}
%
The master project will presumably have the duration of 13 \acp{cw}, which are divided into:

\begin{itemize}
    \item \acsp{cw} 44-45: Software requirements analysis.
    \item \acs{cw} 46: Software architecture design.
    \item \acs{cw} 47: Design of software interfaces.
    \item \acs{cw} 48: Implementation and tests of software drivers.
    \item \acs{cw} 49: Motor Current Extremes Definition.
    \item \acsp{cw} 50-52: Implementation and tests of program logic.
    \item \acsp{cw} 01-03: Paper writing.
    \item \acsp{cw} 04-05: Final review and submission of the paper.
\end{itemize}

Each calendar week will approximately consist of $\frac{150\unit{h}}{14} \approx 10.7$ hours of work.
